# 우진이의 이야기 도서관

GitHub Pages용 전자 동화책입니다.

## 업로드 방법

ZIP을 풀고 아래 파일/폴더를 저장소 루트에 그대로 덮어씌우세요.

- `index.html`
- `style.css`
- `script.js`
- `data.js`
- `assets/`

## 수정 내용

- 이미지 엑박 방지: `data.js` 경로와 실제 이미지 폴더를 모두 일치시켰습니다.
- `제주 바다 이야기` 클릭 오류 수정: 모든 책 카드가 터치/클릭으로 열리게 수정했습니다.
- 기차 여행 + 제주도 여행 Part 1 + 제주 바다 이야기 통합
